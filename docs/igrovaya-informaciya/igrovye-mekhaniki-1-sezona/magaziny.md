# 💸 Магазины

Данная механика позволяет торговать с помощью сундуков и бочек

Создание магазинов - прекрасное времяпровождение и удобное строение для развития базы/спавна

::: info
Данная механика не является чем-то подобным /ah или /shop, наоборот, все является максимально приближенным к реальности для большего интереса.
:::
