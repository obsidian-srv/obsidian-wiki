# 💬 Игровой чат

<span class="inline-badge badge-purple">**!пример**</span> - отправляет ваше сообщение в глобальный чат. Его видят <span class="badge-red">**все**</span> игроки на сервере.

<span class="inline-badge badge-purple">**пример**</span> - отправляет ваше сообщение в локальный чат. Его видят <span class="badge-red">**все игроки в радиусе 100 блоков**</span> от вас.

<span class="inline-badge badge-purple">**/msg /tell ник_игрока пример**</span> - отправит ваше сообщение только указанному игроку.

<span class="inline-badge badge-purple">**/r пример**</span> - отправит ваше сообщение игроку, с которым у вас была переписка в последний раз.
