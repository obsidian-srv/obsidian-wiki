# 🗺️ Карта мира

<span class="hover-text-test">Многим не хватает функционала модов и они просят большее</span>

На сервере установлен прекрасный плагин DynMap, который позволяет смотреть на карту мира в прямом времени вместе с игроками в сети

[**Ссылка на карту сервера**](http://map.obsidianmc.ru:20372/)
<figure><img src="/dynmap1.png" alt=""><figcaption></figcaption></figure>

На данный момент прогружена территория радиусом **5000 блоков** от спавна, для большего исследовательского интереса

## Маркеры

На данный момент доступна всего 1 команда для взаимодействия с картой - <span class="inline-badge badge-purple">**/marker**</span>
Вводится она так: <span class="inline-badge badge-purple">**/marker <имя_маркера> <иконка_на_карте>**</span>
