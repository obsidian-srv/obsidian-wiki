# 🎤 PlasmoVoice

<span class="hover-text-test">Это дополнение к игре позволяет игрока общаться непосредственно в майнкрафте при помощи особого мода c множеством функций.</span>

Для того чтобы общаться с другими игроками с помощью микрофона, вам нужно [скачать](https://modrinth.com/plugin/plasmo-voice/versions?l=fabric\&l=forge\&g=1.21) и установить этот мод.

## **Команды для PlasmoVoice:**

<span class="inline-badge badge-purple">**/vrc**</span> - переподключит вас к PlasmoVoice в случае проблем

<span class="inline-badge badge-purple">**/groups browse**</span> - просмотр всех групп сервера\
<span class="inline-badge badge-purple">**/groups create name: название_группы password: пароль_группы**</span> - создать группу\
<span class="inline-badge badge-purple">**/groups join uuid_группы**</span> - присоединиться к группе \
<span class="inline-badge badge-purple">**/groups info**</span> - информация о группе, в которой состоите \
<span class="inline-badge badge-purple">**/groups invite**</span> _ник\_игрока_ - пригласить кого-то в группу, в которой состоите \
<span class="inline-badge badge-purple">**/groups leave**</span> - выйти из группы, в которой состоите \
<span class="inline-badge badge-purple">**/groups delete**</span> - удалить группу, в которой состоите _(доступно если вы ее владелец)_ \
<span class="inline-badge badge-purple">**/groups set**</span> - сменить пароль/имя группы, в которой состоите _(доступно если вы ее владелец)_ \
<span class="inline-badge badge-purple">**/groups transfer ник_игрока**</span> - передать права группы, в которой состоите _(доступно если вы ее владелец)_ \
<span class="inline-badge badge-purple">**/groups unset password**</span> - убрать пароль в группе, в которой состоите _(доступно если вы ее владелец)_

## Ого, кастомные пластинки!
На сервере есть возможность загружать свои звуки на пластинки и слушать их через проигрыватель с друзьями.

**Команды для пластинок:**

<span class="inline-badge badge-purple">**/disc burn ссылка_на_звук/видео**</span> - записать на специальный музыкальный диск, звук с видео \
<span class="inline-badge badge-purple">**/disc erase**</span> - превратит "Диск для записи" или диск с музыкой обратно в обычную пластинку
