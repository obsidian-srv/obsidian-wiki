# ❓ Как зайти на сервер?

<span class="hover-text-test">На прошлых сезонах, одного неизвестного сервера, игроки испытали проблемы со входом. С этого момента Салат старается сделать как можно больше способов зайти на сервер, но это все лишь легенды...</span>

## 1. Вступить в Discord сервер

Там вы сможете найти **IP** сервера и познакомиться с другими людьми

## 2. Скопировать IP и ввести его в игре

- `Java edition`  
  Сетевая игра → Добавить → Вводишь любое название сервера → Адрес сервера: `ip сервера` → Готово!

- ~~`Bedrock/Windows 10 edition`~~ <span class="badge-red">НЕНАВИЖУ БЕДРОК ИГРОКОВ ГОРЕТЬ ИМ В АДУ СУКАААААААААААА</span>  
  ~~Играть → Серверы → Добавить сервер → Вводишь любое имя сервера → Адрес сервера: `ip сервера` → Порт: `25587` → Сохранить!~~

## 3. Зайти на сервер

Теперь вам нужно залогиниться и отправить полученный код Discord боту `@ObsidianSRV` в личные сообщения.

## 4. Почти получилось!

Осталось перезайти на сервер и вы попали в главный мир
