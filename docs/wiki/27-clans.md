# 🏤 Кланы

<span class="hover-text-test">Кланы - прекрасная вещь, которая может объединить множество людей с разными интересами в один общий коллектив</span>

Для того чтобы создать клан приходит на помощь бот сервера

В канале [Кланы](https://discord.com/channels/1262483802941296780/1526954780893904946) есть форум "Создание клана"  
1. Нажимаешь на кнопку "Создать клан" 
2. Указываешь название, описание, фотографию по желанию
  
Создастся новый форум с кланом, где игроки могут подать заявку в клан, а участники управлять или выйти с клана

:::info
Данный клан будет отображен в игровом профиле!
:::
