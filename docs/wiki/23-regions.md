# 🏳️ Регионы

<span class="hover-text-test">Устал от вечной путанницы где и кто живет? Надоело что твою территорию вечно ломают из-за недопонимания?</span>


Регионы - помеченные места в мире пренадлежащие каким-то людям в виде квадрата радиусом 50 блоков

## Как создать свой регион:
1. Поставьте табличку
2. Напишите на 1 строке [регион]
3. Напишите на 2 строке название региона

Если все верно - высветится сообщение о созданном регионе и табличка поменяет свой цвет

Смена названия на табличке - сменит название региона
## Команды
Для управления регионами создана команда <span class="inline-badge badge-purple">**/region**</span>  
<span class="inline-badge badge-purple">**/region list**</span> - покажет ваши регионы  
<span class="inline-badge badge-purple">**/region info <название региона>**</span> - покажет информацию о регионе  
<span class="inline-badge badge-purple">**/region description <название региона> <описание>**</span> - поставит описание региону

:::info
Также регионы показываются на онлайн карте сервера
:::
