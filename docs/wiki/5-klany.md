# 🏤 Кланы

<span class="hover-text-test">Кланы - прекрасная вещь, которая может объединить множество людей с разными интересами в один общий коллектив</span>

Явление кланов - событие не из новых, но у большинства серверов отсутствует система кланов, из-за чего игроки не могут понять в каком клане состоит игрок.

## **Команды кланов:**
<span class="inline-badge badge-purple">**/clan create имя_клана**</span> - Создать клан  
<span class="inline-badge badge-purple">**/clan invite ник_игрока**</span> - Пригласить в клан  
<span class="inline-badge badge-purple">**/clan accept ник_игрока**</span> - Принять запрос в клан  
<span class="inline-badge badge-purple">**/clan kick ник_игрока**</span> - Выгнать игрока с клана  
<span class="inline-badge badge-purple">**/clan stats**</span> - Топ 10 кланов  
<span class="inline-badge badge-purple">**/clan stats имя_клана**</span> - Статистика клана  
<span class="inline-badge badge-purple">**/clan leave**</span> - Покинуть клан  
<span class="inline-badge badge-purple">**/clan delete**</span> - Удалить клан  
<span class="inline-badge badge-purple">**/clan list**</span> - Список игроков в клане  
<span class="inline-badge badge-purple">**/clan rename имя_клана**</span> - Переименовать клан  

## Цветное название клана

Чтобы сделать цветным название клана достаточно поставить [**&код\_цвета**](https://minecraft.tools/ru/color-code.php) перед названием самого клана
